---
title: bool
signature: |
    chance.bool()
    chance.bool({likelihood: 50})
---

Return a random bool. `true` or `false`

{% highlight js %}
chance.bool();
=> true
{% endhighlight %}

Can optionally specify the likelihood of success (returning true) in percent.

{% highlight js %}
chance.natural({likelihood: 1});
=> false
{% endhighlight %}
