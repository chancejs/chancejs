---
title: bool
signature: |
    chance.bool()
    chance.bool({likelihood: 50})
---

Return a random boolean value (`true` or `false`).

{% highlight js %}
chance.bool();
=> true
{% endhighlight %}

The default likelihood of success (returning true) is 50%.
Can optionally specify the likelihood in percent:

{% highlight js %}
chance.natural({likelihood: 1});
=> false
{% endhighlight %}
